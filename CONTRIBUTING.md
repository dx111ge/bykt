# Contributing to BYKT ITSM

Thank you for your interest in contributing to BYKT ITSM! We welcome all contributions, from bug reports and feature requests to code and documentation improvements.

## Code of Conduct

To ensure a welcoming and inclusive community, this project is governed by a [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## How to Contribute

If you'd like to contribute, please follow these steps:

1.  **Find an issue to work on:** Look for open issues on our GitHub repository. If you have a new idea, please open an issue first to discuss it.
2.  **Fork the repository** and create a new branch for your feature or bug fix.
3.  **Make your changes.** Please ensure your code follows the project's existing style and conventions.
4.  **Write tests** for your changes to ensure they work as expected and don't break anything.
5.  **Submit a pull request.** In your pull request description, please explain the changes you've made and link to any relevant issues.
